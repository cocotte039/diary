# M5-T1: 旧識別子 grep 残留ゼロ化

## 変更対象
- プロジェクト全体

## 実装概要
- `LINES_PER_PAGE` / `splitAtLine30` / `LINES_PER_VOLUME` を rg で検索
- 残留があれば新識別子へ置換（コメントも含む）
- CSS の `--lines-per-page` はコメント内での説明に LINES_PER_PAPER への参照更新を伴うが、変数名自体は残して可（同期コメント明記）

## 受入条件
- 🔵 `rg -n 'LINES_PER_PAGE|splitAtLine30|LINES_PER_VOLUME' src/` で 0 件（CSS `--lines-per-page` は OK、コメントでの言及は OK）
