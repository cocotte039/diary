# M1-T1.1 — notebook.css の罫線均一化

## 変更対象

- `src/styles/notebook.css`

## 実装概要

1. `.notebook-surface` の `background-image` を 2 層 → 1 層に縮小
2. レイヤー1（15 行ごと強調罫線 / `--color-page-divider`）を完全削除
3. レイヤー2（通常罫線 / `--color-rule`）のみ残す
4. `background-size` も 1 行分のみに簡素化
5. コメントを「1 層構成」の説明に書き換え

## 変更後コード

```css
.notebook-surface {
  font-family: var(--font-family);
  font-size: var(--font-size);
  line-height: var(--line-height);
  letter-spacing: var(--letter-spacing);
  color: var(--color-text);
  background-color: var(--color-bg);

  /*
   * 罫線: 1 行ごとに通常罫線（--color-rule）。
   * repeating-linear-gradient で line-height と同期した横線を描く。
   * 変更する場合は background-size と line-height を同期させること。
   * （旧実装にあった 15/30/45 行目の強調罫線レイヤーは仕様変更により廃止）
   */
  background-image: repeating-linear-gradient(
    to bottom,
    transparent 0,
    transparent calc(var(--line-height-px) - 1px),
    var(--color-rule) calc(var(--line-height-px) - 1px),
    var(--color-rule) var(--line-height-px)
  );
  background-size: 100% var(--line-height-px);
  background-attachment: local;
  padding: 0 var(--padding-page);
  background-position: 0 0;
}
```

`.notebook-textarea` と `.notebook-reader` は変更なし。
