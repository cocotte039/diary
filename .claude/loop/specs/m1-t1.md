# T1.1: プロジェクトセットアップ

## 目的
React + Vite + TypeScript のプロジェクトを作成し、PWA基盤、ルーティング、CSS変数、フォント読み込みの初期設定を完了する。

## 変更対象
- `diary/package.json` — 依存定義
- `diary/vite.config.ts` — Vite + PWAプラグイン設定
- `diary/tsconfig.json` — TypeScript設定
- `diary/index.html` — Google Fonts `<link>`、meta設定
- `diary/src/main.tsx` — エントリポイント
- `diary/src/App.tsx` — HashRouter + ルート定義
- `diary/src/styles/global.css` — CSS変数（配色・フォント・行間）

## 詳細仕様

### プロジェクト作成
```bash
npm create vite@latest diary -- --template react-ts
cd diary
npm install idb react-router-dom
npm install -D vite-plugin-pwa
```

### vite.config.ts
- `VitePWA({ registerType: 'autoUpdate' })` の基本設定のみ（詳細はPhase 3）

### App.tsx — ルート定義
```typescript
import { HashRouter, Routes, Route } from 'react-router-dom';
// 4画面: /, /bookshelf, /read/:volumeId/:pageNumber, /settings
```

### index.html
- Google Fonts: `<link href="https://fonts.googleapis.com/css2?family=Klee+One&display=swap" rel="stylesheet">`
- viewport meta: `width=device-width, initial-scale=1, maximum-scale=1`（iOS ズーム防止）

### global.css — CSS変数
```css
:root {
  --color-bg: #1c1c20;
  --color-text: #e8e0d4;
  --color-rule: rgba(255, 255, 255, 0.08);
  --color-page-divider: rgba(255, 255, 255, 0.15);
  --color-accent: #3a3545;
  --font-family: 'Klee One', sans-serif;
  --font-size: 16px;
  --line-height: 1.8;
  --letter-spacing: 0.02em;
}
```
