## 実装概要（3-5行）
1. `DateIcon.tsx` を作成し上記 SVG をそのまま返す関数コンポーネント `export default function DateIcon() { return (<svg ...> ... </svg>); }`。
2. props は不要、サイズは 16x16 固定（必要なら className を受けてスタイル上書きできるよう `props.className` を透過）。
3. アクセシビリティ: SVG に `aria-hidden="true"` を付与（ボタン側で aria-label を持つ）。
4. stroke 属性を SVG 要素レベルに集約して各 line/rect に継承。
