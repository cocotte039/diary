# M3-T2: useEditorCursor の scrollTop 宛先変更

## 変更対象
- `src/hooks/useEditorCursor.ts`（存在する場合）
- `src/features/editor/EditorPage.tsx`（ref 渡しが必要な場合）

## 実装概要
- `useEditorCursor` 内で textarea.scrollTop を設定している箇所があるか grep で確認
- あれば、スクロールコンテナが `.surface` に変わったため、surface の ref（または `textarea.closest('[data-testid="editor-surface"]')`）に scrollTop を書くよう変更
- 代替案: `el.scrollIntoView({ block: 'center' })` で cursor 行を可視領域に
- 🟡 具体実装は `useEditorCursor.ts` の内容を見て判断（推奨: surface ref を引数追加）

## 受入条件
- 🔵 カーソル復元時に `.surface` がスクロールしてカーソル行が可視範囲に入る
- 🔵 既存の Editor テストが引き続き緑
- 🟡 textarea.scrollTop を触るコードが残っていない（または無害化されている）
