# T8-4.1: constants.ts に長押し定数追加

## 変更対象
- `src/lib/constants.ts`
- `src/lib/constants.test.ts`

## 実装概要
VolumeCard の長押し削除で使う 2 定数を追加する。

## 追加定数

```ts
/** 長押し検知までの時間 (ms)。iOS context menu (500ms〜) と同等で自然に感じる値。 */
export const LONG_PRESS_MS = 500;

/** 長押し中に指が動いてもキャンセルしない閾値 (px)。手ブレ許容 */
export const LONG_PRESS_MOVE_TOLERANCE_PX = 10;
```

## 受入条件
- 🔵 `LONG_PRESS_MS === 500`
- 🔵 `LONG_PRESS_MOVE_TOLERANCE_PX === 10`
- 🔵 constants.test.ts で値 assertion 追加

## テスト方針
既存 `constants.test.ts` のパターンに倣って追加:

```ts
it('LONG_PRESS_MS', () => {
  expect(LONG_PRESS_MS).toBe(500);
});

it('LONG_PRESS_MOVE_TOLERANCE_PX', () => {
  expect(LONG_PRESS_MOVE_TOLERANCE_PX).toBe(10);
});
```

## 配線検証
- 使用元は M8-4.2 の VolumeCard.tsx
- `grep -rn "LONG_PRESS_MS" src/` で constants.ts + VolumeCard.tsx の 2 箇所ヒットを目標

## 注意事項
- constants.ts の既存 export 群 (LINES_PER_PAGE / PAGES_PER_VOLUME / SWIPE_THRESHOLD_PX 等) と同じ書式で追加
