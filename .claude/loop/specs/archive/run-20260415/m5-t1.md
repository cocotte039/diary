## テスト書き換え

共通ヘルパ追加:

```tsx
async function openMenu() {
  fireEvent.click(screen.getByRole('button', { name: 'メニューを開く' }));
}
```

`NewVolumeCard` 経由のテストをメニュー経由に置換:

```tsx
it('メニューの「新しいノート」項目が表示される', async () => {
  await ensureActiveVolume();
  renderPage();
  await screen.findByRole('link', { name: /ノート 1/ });
  await openMenu();
  expect(screen.getByRole('menuitem', { name: '新しいノート' })).toBeInTheDocument();
});

it('メニューの「新しいノート」を押すと confirm → 冊が増える', async () => {
  await ensureActiveVolume();
  const confirmSpy = vi.spyOn(window, 'confirm').mockReturnValue(true);
  renderPage();
  await screen.findByRole('link', { name: /ノート 1/ });
  await openMenu();
  fireEvent.click(screen.getByRole('menuitem', { name: '新しいノート' }));
  await screen.findByRole('link', { name: /ノート 2/ });
  confirmSpy.mockRestore();
});
```

設定リンクのテストもメニュー経由に書き換え（既存の「設定」へのナビゲーションテストが存在する場合）。
