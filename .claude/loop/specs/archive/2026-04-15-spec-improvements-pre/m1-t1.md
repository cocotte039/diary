# M1-T1: constants.ts 定数差し替え

## 変更対象
- `src/lib/constants.ts`

## 実装概要
- `LINES_PER_PAGE` 削除
- `CHARS_PER_PAGE = 1200` 新設（JSDoc: 1ページの文字上限、text.length ベース、改行含む）
- `LINES_PER_PAPER = 60` 新設（JSDoc: 罫線描画用の視覚上の行数、CHARS_PER_PAGE とは独立、CSS `--lines-per-page` と同期）
- `PAGES_PER_VOLUME = 50 → 60` に変更
- `LINES_PER_VOLUME` は grep して参照有無確認。未使用なら削除、使用中なら `CHARS_PER_VOLUME = CHARS_PER_PAGE * PAGES_PER_VOLUME` に置換
- `LINE_HEIGHT_PX` / `FONT_SIZE_PX` / `LINE_HEIGHT_EM` / `HEADER_HEIGHT_PX` は維持

## 受入条件（確信度）
- 🔵 `CHARS_PER_PAGE === 1200`
- 🔵 `LINES_PER_PAPER === 60`
- 🔵 `PAGES_PER_VOLUME === 60`
- 🔵 `LINES_PER_PAGE` エクスポートが存在しない
- 🟡 `LINES_PER_VOLUME` は参照次第（未使用なら削除）

## 注意
- JSDoc は既存の書き方（M9/M10 等のタグ参照）を踏襲
- CSS `--lines-per-page` との同期コメントを入れる
