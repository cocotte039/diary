# M2-T4: textarea 高さ useLayoutEffect 追従 + ensurePaperHeight 撤去

## 変更対象
- `src/features/editor/EditorPage.tsx`

## 実装概要
- `ensurePaperHeight` 関数定義を削除
- `handleChange` 内の `ensurePaperHeight()` 呼び出しを削除
- 初回ロード / text 変更時に textarea 高さを scrollHeight に追従:
  ```ts
  useLayoutEffect(() => {
    const ta = textareaRef.current;
    if (!ta) return;
    ta.style.height = 'auto';
    ta.style.height = `${ta.scrollHeight}px`;
  }, [text, ready]);
  ```
- 紙高さの下限は CSS 側（M3-T1 の min-height）で保証する前提

## 受入条件
- 🔵 text 入力で textarea の height が内容に追従
- 🔵 空ページでも CSS min-height により罫線が描画される（M3 と連動）
- 🔵 `ensurePaperHeight` が存在しない
