# M3-T3: 罫線・min-height 動作確認

## 変更対象
- 検証のみ。必要なら `src/styles/notebook.css` / `src/styles/global.css` の変数更新

## 実装概要
- `--page-height-px` が `LINES_PER_PAPER * LINE_HEIGHT_PX` = 60 * 28.8 = 1728 になることを確認
- CSS 変数宣言箇所（global.css or notebook.css）で `--lines-per-page: 60` のコメントを LINES_PER_PAPER と同期することを明記
- 罫線の `background-attachment: local` が textarea height 可変でも機能することをテストで確認（罫線は内容と一緒にスクロール＝surface スクロールで textarea が動くので同期する）

## 受入条件
- 🔵 空ページで罫線 60 本
- 🔵 満量ページ（1200字）でも罫線が破綻しない
- 🔵 surface スクロールに罫線が追従（textarea 全体が surface 内で動くため、罫線も一緒に動く）
