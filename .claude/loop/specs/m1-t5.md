# M1-T5: 25行強調罫線CSS追加

## 目的

25 行ごとに罫線をごくわずかに濃くする視覚レイヤーを追加し、ボリューム感を静かに表現する。

## 変更対象

- `src/styles/notebook.css`

## 依存する既存コード

- CSS 変数 `--line-height-px`（1 行の高さ、`LINE_HEIGHT_PX` と同期）
- CSS 変数 `--color-rule`（既存罫線色、通常は `rgba(255, 255, 255, 0.08)` 相当）
- `.notebook-surface` セレクタの `background-image` / `background-size`

## 実装概要

`src/styles/notebook.css` L25-31 付近の `background-image` を 2 レイヤー重ねに変更する:

```css
.notebook-surface {
  /* 他のプロパティは維持 */
  background-image:
    /* 25 行ごとの強調罫線（上レイヤー） */
    repeating-linear-gradient(
      to bottom,
      transparent 0,
      transparent calc(25 * var(--line-height-px) - 1px),
      rgba(255, 255, 255, 0.14) calc(25 * var(--line-height-px) - 1px),
      rgba(255, 255, 255, 0.14) calc(25 * var(--line-height-px))
    ),
    /* 1 行ごとの通常罫線（下レイヤー、既存） */
    repeating-linear-gradient(
      to bottom,
      transparent 0,
      transparent calc(var(--line-height-px) - 1px),
      var(--color-rule) calc(var(--line-height-px) - 1px),
      var(--color-rule) var(--line-height-px)
    );
  background-size:
    100% calc(25 * var(--line-height-px)),
    100% var(--line-height-px);
  /* background-attachment は既存維持（通常は local または scroll） */
}
```

**色の根拠**: 既存 `--color-rule = 0.08` と `--color-page-divider = 0.15` の中間値 `0.14`。視覚階層を崩さず強調だけを追加。

## 注意事項

- `background-attachment` は既存設定（`local` or `scroll`）を**変更しない**。スクロール同期の挙動を維持。
- `--color-rule` 変数が未定義なら、実ファイルで実際の値を確認して反映する（plan.md 段階での推定値）。
- 色値 0.14 は初期値。Verify で 0.12 / 0.16 と比較して違和感があれば微調整可（T7 で判断）。
