# T8-1.3: BookshelfPage ヘッダー固定 + 縦スクロール修正

## 変更対象
- `src/features/bookshelf/BookshelfPage.tsx`
- `src/features/bookshelf/BookshelfPage.module.css`

## 依存
- T8-1.1（.app-header クラスが global.css にあること）

## 実装概要
本棚ヘッダーを上部固定化（app-header 利用）、`.root` を `height: 100dvh` に変えてスクロールコンテナ化、`padding-top` でヘッダー分の逃げを確保する。

## 差分イメージ

### BookshelfPage.tsx
```tsx
// before
<header className={styles.header}>
// after
<header className={`app-header ${styles.header}`}>
```

### BookshelfPage.module.css
```css
/* before */
.root {
  min-height: 100dvh;
  padding: max(1rem, env(safe-area-inset-top)) var(--padding-page)
    max(1rem, env(safe-area-inset-bottom));
  overflow-y: auto;
}
.header {
  display: flex;
  justify-content: space-between;
  align-items: baseline;
  margin-bottom: 1.5rem;
  font-family: var(--font-family-ui);
}

/* after */
.root {
  height: 100dvh;
  padding-top: calc(var(--header-height) + env(safe-area-inset-top, 0px) + 1rem);
  padding-left: var(--padding-page);
  padding-right: var(--padding-page);
  padding-bottom: max(1rem, env(safe-area-inset-bottom));
  overflow-y: auto;
}
.header {
  /* app-header 側で賄うため空に近い状態。タイトル位置調整のみ残す */
  align-items: baseline; /* h1 と Link のベースライン揃え */
}
```

## 受入条件
- 🔵 BookshelfPage `<header>` に `app-header` クラスが含まれる
- 🔵 `.root` に `height:100dvh` + `overflow-y:auto` が設定される（className アサートで間接検証）
- 🔵 カード数を増やしたときに縦スクロールで見られる（実機確認）
- 🔵 第 1 冊カードがヘッダーに隠れない（`padding-top` 分の余白）
- 🔵 既存 BookshelfPage.test.tsx が全通過

## テスト方針
- 新規: ヘッダー要素の className に `app-header` が含まれることを確認する 1 ケース追加
- 既存テストは壊さない

## 注意事項
- `header` の `margin-bottom: 1.5rem` は削除（fixed でフローから外れるため）
- `.header button { opacity:0.3 ... }` 等の既存副次スタイルは残す
- カレンダー表示時のレイアウト（`.calendarToggle` + `<Calendar/>`）は `.root` の padding 内で正常に収まること
