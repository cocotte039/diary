# M4-T1: EditorPage.tsx にプログレスバー要素追加

## 変更対象
- `src/features/editor/EditorPage.tsx`

## 実装概要
- `<header>` と `<div className={styles.surface}>` の間に挿入:
  ```tsx
  {(() => {
    const pct = Math.min(100, Math.round((text.length / CHARS_PER_PAGE) * 100));
    return (
      <div
        className={styles.progress}
        role="progressbar"
        aria-label="ページの残量"
        aria-valuemin={0}
        aria-valuemax={100}
        aria-valuenow={pct}
        data-testid="page-progress"
      >
        <div className={styles.progressFill} style={{ width: `${pct}%` }} />
      </div>
    );
  })()}
  ```
- aria-valuetext は付けない（静けさ）
- 色変更・アニメーションなし

## 受入条件
- 🔵 role=progressbar 要素がヘッダー直下に常時描画
- 🔵 text.length=0 → aria-valuenow=0、=600 → 50、=1200 → 100、=1300 → 100（clamp）
- 🔵 aria-valuemin=0, aria-valuemax=100, aria-label 付与
