# M7-T2: EditorPage 本文と罫線の整合

## 目的
EditorPage のヘッダー（fixed）が本文 1 行目と重ならないよう padding-top を入れ、罫線と本文ベースラインが揃うよう background-position を調整する。safe-area-inset 対応も含める。

## 変更対象
- ファイル:
  - `/home/cocotte/dev/diary/src/features/editor/EditorPage.module.css`
  - `/home/cocotte/dev/diary/src/features/editor/EditorPage.tsx`（必要ならヘッダーを `position: fixed` に）
- CSS 変更:
  - textarea/本文コンテナ: `padding-top: calc(var(--header-height) + env(safe-area-inset-top, 0px));`
  - 罫線背景: `background-position: 0 calc(env(safe-area-inset-top, 0px));`
  - ヘッダー: `position: fixed; top: 0; left: 0; right: 0; height: var(--header-height); z-index: 2; background: var(--color-bg);`

## 依存する既存コード
- T7.1 `--header-height`
- `/home/cocotte/dev/diary/src/styles/notebook.css` `.notebook-surface` の `background-*` 設定

## 実装概要（3-5行）
1. `EditorPage.module.css` で `.header { position: fixed; ... height: var(--header-height); background: var(--color-bg); z-index: 2; }` を定義。
2. `.surface` (textarea ラッパ) に `padding-top: calc(var(--header-height) + env(safe-area-inset-top, 0px))` を設定。
3. `.surface.notebook-surface` で `background-position: 0 calc(env(safe-area-inset-top, 0px))` を override（本文テキストと罫線が一緒に下がる）。
4. z-index: header > surface、background は header 下に本文が透けないよう不透明。

## 注意事項
- `env(safe-area-inset-top)` は jsdom では 0 として解決。実機目視が最終判断。
- `notebook-surface` のグローバル background 設定と CSS Module の `.surface` 指定の詳細度に注意（:global / @apply 不可のため CSS Module 内で `:global(.notebook-surface).surface { ... }` のような上書きが必要なら記述する）。
- 罫線と本文が一緒に下がる＝罫線の反復パターン開始位置をヘッダー分シフトする。背景サイズは維持。
