## 完了サマリー

- 実変更:
  - `src/features/bookshelf/Calendar.tsx` + `Calendar.module.css` (前後月、曜日行、日付セル、ドット)
  - `db.findPageByDate` / `db.getDateSetInMonth`
  - BookshelfPage に折りたたみ式カレンダートグル
- 判断:
  - 自作カレンダー（ライブラリ不要）。曜日配列 `['日','月',...]`、月初オフセット計算でセル生成。
  - findPageByDate: 同日完全一致を優先、無ければ最近日（差分の絶対値最小）を返す。
  - getDateSetInMonth: prefix マッチ `YYYY-MM` で全Pageを走査。Page 数が増えた場合の最適化は今後。
- 注意:
  - タイムゾーンは createdAt (ISO UTC) と `YYYY-MM-DD` 文字列比較。JST で 23時作成なら翌日 UTC で記録される可能性がある。v1 では許容、後で localized date key を検討。
