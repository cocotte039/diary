# T8-1.1: global.css に .app-header 共通クラス追加

## 変更対象
- `src/styles/global.css`

## 実装概要
3 画面（Editor / Bookshelf / Settings）で共通化するヘッダー用クラスを追加する。
左右パディングは `max(1rem, env(safe-area-inset-*))` に統一。既存 `.app-header-link` の下に追記する。

## 新規 CSS

```css
.app-header {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  height: calc(var(--header-height) + env(safe-area-inset-top, 0px));
  padding-top: env(safe-area-inset-top, 0px);
  padding-left: max(1rem, env(safe-area-inset-left));
  padding-right: max(1rem, env(safe-area-inset-right));
  display: flex;
  justify-content: space-between;
  align-items: center;
  z-index: 2;
  background-color: var(--color-bg);
  font-family: var(--font-family-ui);
}
```

## 受入条件
- 🔵 `.app-header` セレクタが global.css に存在する
- 🔵 position: fixed で top 0 に固定される
- 🔵 左右 padding が `max(1rem, ...)` 形式

## テスト方針
- CSS のみの変更。下流テストで間接検証。

## 注意事項
- 既存 `.app-header-link` はそのまま維持（EditorPage / BookshelfPage / SettingsPage で使用中）
